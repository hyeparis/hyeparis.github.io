About

Joowelry is an E-Commerce store using ReactJS, Node.js, HTML, CSS, JavaScript and MongoDB. Joowelry is an online jewelry store which displays details of beautiful jewelry products and provides customers with the easiest method to shop the jewelry by offering a shopping cart and shipping. The main purpose of this application is that providing services to not only customers but also admin to help the authorized user to manage the store effiently and conviniently.
