Joowelry

Joowelry is an E-Commerce store using ReactJS, Node.js, HTML, CSS, JavaScript and MongoDB. Joowelry is an online jewelry store which displays details of beautiful jewelry products and provides customers with the easiest method to shop the jewelry by offering a shopping cart and shipping. The main purpose of this application is to let not only users but also an administrator use the application effiently and conviniently.

Software Toolset

Features

Main Page

Users can register or sign-in using SignIn button and access their shopping cart by using cart icon on the top-right corner of the page. Main page displays a list of products and promotions.

Register & Sign-in

A new user can register using an email address and a registered user can log into the app and add items to a shopping cart to proceed to check out.

Manage a cart & Order an item

Users can manage their shopping cart. If users click on 'Delete' button, item will be deleted from the cart. When users click on 'Proceed-to-Checkout' button, it takes users to order-page that requires delivery information and payment using paypal. After all required fields are filled, the order process is completed.

Admin feature

Admin has an option of managing the application. By clicking a pencil icon, admin can apply CRUD(create, read, update, delete) operation.
